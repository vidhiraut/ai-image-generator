# AI Image Generator using ComfyUI

## Overview

This project generates images from text prompts using Stable Diffusion and ComfyUI.

It uses a node-based workflow to convert prompts into images.

## Features

- Custom image generation pipeline
- HuggingFace model integration
- Automated prompt-to-image generation
- Modular workflow architecture

## Workflow

Prompt → CLIP Encode → KSampler → VAE Decode → Save Image

## Tools Used

- Python
- ComfyUI
- Stable Diffusion
- HuggingFace
